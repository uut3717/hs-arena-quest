💾 데이터 관리

📤 내보내기 — JSON 파일로 저장 후 다른 기기에서 불러올 수 있습니다.
📥 가져오기 — 기존 JSON 파일을 선택합니다. 현재 데이터를 덮어씁니다.
💡 다기기 동기화 방법
1. 이 기기에서 내보내기 → JSON 파일을 카카오톡/메일로 전송
2. 다른 기기 브라우저에서 이 앱 열기 → 가져오기
3. 또는 Google Drive에 JSON 저장 후 공유 링크로 접근
